🛡 About PhishCatcher

PhishCatcher is a web-based email phishing detection system developed using Python, Flask, HTML, CSS and SQLite.

The system analyzes uploaded .eml email files and detects phishing indicators using multiple security checks.

Key Features

  • ✔ Email Header Analysis
  • ✔ URL Extraction & Analysis
  • ✔ SPF, DKIM & DMARC Detection
  • ✔ Threat Score Calculation
  • ✔ Threat Level Classification
  • ✔ Suspicious Keyword Detection
  • ✔ Attachment Detection
  • ✔ IP Address Extraction
  • ✔ Suspicious Sender Detection
  • ✔ SQLite Scan History
  • ✔ Export Reports (CSV & PDF)

This project demonstrates how phishing emails can be analyzed automatically to help users identify potential security threats.
