# PhishCatcher

An Intelligent Email Phishing Detection and Analysis System developed using Flask.

## Features
- Email (.eml) analysis
- URL analysis
- SPF, DKIM and DMARC checking
- Threat scoring
- Dashboard with scan statistics
- Scan history
- PDF and CSV report export
- Secure login with password hashing

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
